# Roadmap de produto — Te Organiza

Features por módulo (após billing). Prioridade sugerida: P0 próximo, P1 depois, P2 later.

## Painel

- P0: strip de patrimônio líquido
- P1: digest “o que mudou esta semana”
- P2: atalhos a partir dos Insights

## Insights (Pro)

- P0: regras atuais + categorias / vs mês passado / sobra → meta
- P1: insights por dia da semana; % fixos vs variáveis
- P2: resumo semanal por e-mail (Pro)

## Transações

- P0: filtros por mês / categoria / conta
- P1: detecção de duplicata
- P2: categorização em lote

## Recorrentes

- P0: calendário de vencimentos
- P1: unificar com recorrências de cartão
- P2: auto-confirmar regras simples

## Cartões

- P1: fatura do mês
- P1: % do limite utilizado
- P2: “pagar fatura” a partir de uma conta

## Dívidas

- P1: plano avalanche / snowball
- P1: juros economizados se pagar X a mais
- P2: tip “dívida vs investir” (liga ao Insights)

## Metas

- P1: ETA com sobra do mês
- P2: auto-aporte sugerido
- P2: ligar meta ↔ investimento

## Investimentos (Pro)

- P1: pizza de alocação
- P1: evolução de patrimônio
- P2: histórico de aportes

## Renda

- P1: gauge renda vs custos fixos
- P2: estimativa simples freela / impostos

## Importar

- P0: reexibir no nav quando liberar oficialmente
- P1: OFX + mapeamento de colunas
- P2: presets por banco (sem prometer Nubank sync)

## Relatórios

- P1: sair do redirect — relatório mensal
- P2: export PDF/CSV; comparação YoY

## Assinatura / billing

- P0: trial 1º mês, MP, cancelar, histórico (feito)
- P1: páginas Termos / Privacidade
- P2: Pix automático / dunning avançado; avaliar Stripe se houver expansão internacional
