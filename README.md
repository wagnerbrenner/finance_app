# Finora

Sistema pessoal para organizar patrimônio, fluxo de caixa, dívidas e metas.

**Tagline:** Suas finanças em ordem.

## Stack

- Next.js (App Router) + React
- Supabase Auth + Postgres (Drizzle)
- Vercel
